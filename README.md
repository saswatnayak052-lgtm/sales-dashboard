# sales-dashboard
Interactive Power BI Sales Dashboard analyzing Super Store data to track sales, profit, quantity, and delivery performance. Includes category, region, ship mode, payment mode analysis with YOY sales and profit trends to support data-driven business decisions

# 2. Project Objective :

1-Analyze overall sales and profit performance

2-Track key business KPIs and trends

3-Enable data-driven sales and operational decisions

# 3. Business Problem :

Businesses struggle to understand which products, regions, and customer segments drive revenue and profit. This project helps identify sales trends, profitability patterns, and performance gaps across multiple dimensions.

# 4. Dataset Information :

1-Order and shipment details

2-Sales, profit, and quantity data

3-Product categories and sub-categories

4-Customer segments

5-Region, state, and payment mode data

(Sample Super Store dataset used for analytical practice.)

# 5. Tools & Technologies Used :

1-Power BI

2-DAX

3-Power Query

4-Excel / CSV

# 6. Key KPIs :

1-Total Sales

2-Total Profit

3-Quantity Sold

4-Average Delivery Time

5-Monthly Sales & Profit (YOY)

# 7. Analysis Performed :

1-Sales and profit by category and sub-category

2-Regional and state-wise sales analysis

3-Segment-wise and payment mode analysis

4-Ship mode performance comparison

5-Year-over-Year (YOY) sales and profit trends

# 8. Dashboard Features :

1-Interactive slicers for region filtering

2-Dynamic KPIs using DAX measures

3-Clear visual hierarchy for business users

4-Clean, professional dashboard design

# 9. Key Insights :

1-Certain categories contribute higher sales but lower profit

2-Regional sales performance varies significantly

3-YOY trends highlight seasonal sales patterns

4-Shipping and payment modes impact delivery and revenue

# 10. Business Impact :

1-Helps managers identify high-performing products and regions

2-Supports sales strategy and inventory planning

3-Improves decision-making through real-time insights
